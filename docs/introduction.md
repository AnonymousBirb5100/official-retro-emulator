# Getting Started

BirdHub Emulator (EmulatorJS) is simple to use once you know the basics.
This doc was assisted by AI.

> [!IMPORTANT]
> This guide covers only basic features. It may not include everything.

---

## Loading Your First Game

1. Open the emulator in your browser: https://anonymousbirb5100.github.io/official-retro-emulator  
   *(Also available on [BirdHub V2 v1.9.7](https://github.com/anonymousbirb5100/birdhub-v2) and later)*

2. Click the gray box to upload your ROM **or** drag and drop your ROM file into it

---

## Using the Bottom Menu

Move your cursor to the bottom of the screen to open the menu.

From here you can:
- Pause the game  
- Adjust volume  
- Change controls  
- Open settings  

---

## Changing Controls

1. Open the bottom menu and click **Controls**
2. Select the button you want to change
3. Press a key or controller button to reassign it

---

## Save States

### Creating a Save State

1. Open the bottom menu
2. Click **Save State**
3. A `.state` file will be downloaded

### Loading a Save State

1. Load your game first  
2. Drag and drop the `.state` file into the emulator  
   **or** click **Load State** in the menu  

> ⚠️ Do not load a save state before loading a game.

---

## Troubleshooting

### Emulator shows a console dropdown and "Load" button

This usually means the file type is not supported (e.g., ZIP files or unsupported ROMs).

---

### Controls not working

Controls are saved per URL. If you switch devices or domains, you may need to reassign them.

---

### Stuck on "Downloading game data"

This is usually caused by a slow connection or device limitations.
